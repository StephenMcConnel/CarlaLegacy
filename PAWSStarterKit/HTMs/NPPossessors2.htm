In the lexicon, mark the affix with the feature genitive.

You need to write a CC table pre-pass that will split the clitic into a separate word. Then add a root entry to your lexicon for that clitic, and add the feature Poss to assure the correct category and genitive case feature for the syntactic parser. [Note that this may be a little more complicated. For example, in English 's can also be the contraction of is or has, so those also need to have root entries and the .ana file will have a three-way ambiguity. The syntactic parser will then disambiguate it. In addition, regular English plurals have a different form for the possessive (e.g. boys'), so the CC table must change ___s' to ___s 's and ___'s to ____ 's.]

In the lexicon, mark this entry with the feature Poss, which will change the category and add the genitive case.
