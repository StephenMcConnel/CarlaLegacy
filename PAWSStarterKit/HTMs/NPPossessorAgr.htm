Nominal Phrase - Possessor Agreement


Any agreement marking on nouns for person, number, gender, animacy and/or class should have already been done in the section on Articles and Demonstratives section.

Are nouns marked with an affix which agrees with the possessor?
No
Yes

In the lexicon, add one or more features as appropriate to the affix entries from the following list. The first feature in the list possessed is used when it is the only affix used and it simply marks that a possessor must be present. It is conceivable that there is a language that has this type of general affix for possession and also has the more specific affixes that mark agreement with the possessor, probably not occuring on the same noun.
Feature Explanation
possessed there must be a possessor
poss_first the possessor must be first person
poss_exclusive the possessor must be first person exclusive
poss_inclusive the possessor must be first person inclusive
poss_second the possessor must be second person
poss_third the possessor must be third person
poss_singular the possessor must be singular number
poss_plural the possessor must be plural number
poss_dual the possessor must be dual number
poss_feminine the possessor must be feminine gender
poss_masculine the possessor must be masculine gender

Note that this possessor agreement, at least in person features, cannot be tested until pronoun possessors are added in section 6 Pronouns and Proper Names.
