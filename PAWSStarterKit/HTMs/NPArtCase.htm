You have a nominative-accusative case system.

  • In the lexicon, mark the articles and demonstratives that go with subjects with the feature nominative and those that go with direct objects with the feature accusative.
  • If you have a set for possessors, add the feature genitive to those entries.
  • If a particular article or demonstrative can show up in any position, do not add any feature.
  • If a particular article or demonstrative can occur as a subject or as a possessor but not as the object of a verb or pre/post-position, add the feature -accusative.
  • If a particular article or demonstrative can occur as a subject or as an object but not as a possessor, add the feature -genitive.
  • If a particular article or demonstrative may occur as an object or as a possessor, but not as a subject, add the feature -nominative.

You have an ergative-absolutive case system.

  • In the lexicon, mark the articles and demonstratives that go with subjects of transitive verbs with the feature ergative and those that go with direct objects and subjects of intransitive verbs with the feature absolutive.
  • If you have a set for possessors, add the feature genitive to those entries.
  • If a particular article or demonstrative can show up in any position, do not add any feature.
  • If a particular article or demonstrative can occur as a subject of transitive verbs or as a possessor but not as an object or as the subject of an intransitive verb, add the feature -absolutive.
  • If a particular article or demonstrative can occur as a subject of any verb or as an object but not as a possessor, add the feature -genitive.
  • If a particular article or demonstrative may occur as an object or the subject of an intransitive verb or as a possessor, but not as the subject of a transitive verb, add the feature -ergative.
