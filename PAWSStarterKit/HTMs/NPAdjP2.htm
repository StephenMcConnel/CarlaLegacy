Can all the adjective phrases appear on either side of the noun?
Yes, there is no restriction
No, there are separate sets for each side

In the lexicon, mark those adjectives which can only occur before the noun with the feature DP-initial and those which can only occur after the noun with the feature DP-final. Do not enter either feature on any adjective which can occur on either side.
