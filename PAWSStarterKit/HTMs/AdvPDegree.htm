Adverbs and AdvP - Degree Words


You need to make sure the adverbs in your lexicon will be mapped to the syntax category Adv. Choose one of the following options, depending on your situation, and check it off when it is completed:

  • If you already have adverbs marked with the category Adv in your lexicon nothing more needs to be done.
  • If you have adverbs marked in your lexicon with a distinct category all their own, but it is different from Adv, go to the Category Mapper and map your morphological category to Adv.
  • If adverbs are mixed in with another category in the lexicon, you need to add the feature Adv to each appropriate entry so the category will be changed.

In the lexicon, mark those which can only occur before the adverb with the feature AdvP-initial and those which can only occur after the adverb with the feature AdvP-final. Do not enter either feature on any degree words which can occur on either side.

In the lexicon, key the features:

  • Deg modifies_temporal for temporal modifiers,
  • Deg modifies_locative for locative modifiers,
  • Deg modifies_manner for manner modifiers, and
  • Deg modifies_reason for reason modifiers
