In the lexicon, mark those quantifiers or numbers which can only occur before the noun with the feature DP-initial and those which can only occur after the noun with the feature DP-final. [Note: if all numbers can only go on one side or the other, rather than entering the feature on every entry in the lexicon, add [DP-initial] or [DP-final] as appropriate to the Num template using the Feature Template Editor.] Do not enter either feature on any quantifier or number which can occur on either side.
