Quantifiers and QP - No Degree Words


In the lexicon, mark this type of quantifier with the feature Q. This will change their category to Q for the syntax. If a particular quantifier can only go with plural nouns, add the feature plural to that entry. Also, mark any words which can modify them with the features Deg modifies_Q. Note that these degree words will most likely not be the same as the ones which can modify an NP above.
