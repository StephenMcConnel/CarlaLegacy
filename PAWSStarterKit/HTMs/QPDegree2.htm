In the lexicon, mark this type of quantifier with the feature Q. This will change their category to Q for the syntax. If a particular quantifier can only go with plural nouns, add the feature plural to that entry. Also, mark any words which can modify them with the features Deg modifies_Q. Those which can only occur before the quantifier also need to have the feature QP-initial and those which can only occur after the quantifier also need the feature QP-final. Do not enter either feature QP-initial or QP-final on any degree words which can occur on either side. Note that these degree words will most likely not be the same as the ones which can modify an NP above.
