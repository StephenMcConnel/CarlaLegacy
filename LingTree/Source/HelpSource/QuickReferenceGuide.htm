LingTree Quick Reference Guide

Key This To Get This
( A new tree level
) To end a tree level
\( To get an open parenthesis
\) To get a closed parenthesis
\G To indicate the node is a gloss node
\L To indicate that the node is a lexical item node
\O To omit the line above a node
\T To show a triangle over the node
/s To cause the rest of a node's text to be shown as subscript
/S To cause the rest of a node's text to be shown as superscript
