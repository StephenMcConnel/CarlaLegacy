Archivos Answer and Result File Information

The PAWS Starter Kit can create several files.

  1. One stores the answers you have given to the questions.
  2. Another stores the automatically generated PC-PATR grammar file.
  3. A third stores an automatically generated comparative grammar write-up file.
  4. A fourth stores an automatically generated practical grammar write-up file.
  5. A fifth stores an automatically generated practical grammar write-up in Spanish file.
  6. Finally, there are many example files it creates so you can test the PC-PATR grammar.
You can set these files by answering the following questions.


Select the outputs you want by checking the box before each one. Please note that at least one writer output must be present. The default writer is the comparative one.

Output the PC-PATR grammar file and test example files.
Output the comparative grammar write-up file.
Output the practical grammar write-up file.
Output the Spanish practical grammar write-up file.
